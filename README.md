# Hangman Game - CodeAlpha Internship

This is a simple text-based Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

# Description
The Hangman Game is a word guessing game where the player tries to guess a hidden word one letter at a time. The player has limited attempts to guess the correct word.

# Objective
To build a basic game using Python that improves logic building and understanding of loops and conditions.
